## What To Send

When opening the request form or emailing the brief, include:

- File type.
- Approximate row count.
- Approximate column count.
- Cleanup type.
- What should be cleaned.
- What must not be changed.
- Payment sender email, if you already paid.

Payment sender email is used only to match a visible PayPal payment to the request.
