## What To Send

When opening the request form or emailing the brief, include:

- File type.
- Approximate row count.
- Approximate column count.
- Cleanup type.
- What should be cleaned.
- What must not be changed.
- Payment sender email, if you already paid.

Payment sender email is used only to match a visible PayPal payment to the request.

## Not Included

No accounting, tax, legal, investment, audit, import troubleshooting, research, data-verification, guaranteed-income, marketplace approval, shipping result, or payment dispute result is included. Do not send private credentials, bank details, card data, or passwords.

If the file is larger than 300 rows or 12 columns, open the request form before paying.
